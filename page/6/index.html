获取视频列表

接口地址：

/channel/video

接受参数和返回信息

接收参数为：
channelId（频道ID，1=动漫）
regionId（地区ID） typeId（分类ID） end（是否完结） sort（排序方式） offset（起始条数） limit（每页条数）
返回信息：
id = 视频ID
title = 标题
sub_title = 副标题
img = 竖版图片地址
img1 = 横版图片地址
add_time = 添加时间
episodes_count = 剧集集数
is_end = 是否完结
业务逻辑：
获取video表中数据
channelId=接收参数（必填项）
status=1
regionId、typeId（根据是否传值，增加筛选条件）
end（n=未完结，y=已完结）
sort（默认时间倒序，episodesUpdateTime=剧集更新时间倒序，comment=评论数倒序）

获取评论列表

/comment/list

  • 接收参数为：
