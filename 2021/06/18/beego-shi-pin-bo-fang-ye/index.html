beego视频播放页

发表于 2021-06-17 更新于 2021-06-21 分类于 beego

页面分为两种情况：

  • 未指定剧集数，默认打开第一集，播放第一集的视频
  • 指定剧集数，打开对应剧集信息

controllers\video.go

routers\default.go

视频详细信息接口

接口地址：/video/info

  • 接收参数为：videoId（视频ID）

  • 返回信息：
    id = ID
    title = 标题
    sub_title = 副标题
    add_time = 添加时间
    img = 竖版缩略图
    img1 = 横版缩略图
    episodes_count = 剧集数
    … …和video表中的数字段是一一对应的

  • 业务逻辑：获取video表中id=接收参数的数据。
    注意：这里没有获取到视频播放地址或者阿里云视频ID

视频剧集列表接口

接口地址：/video/episodes/list

  • 接收参数为：videoId（视频ID）

  • 返回信息：
    id = ID
    title = 剧集标题
    add_time = 添加时间
    num = 第几集
    play_url = 播放地址
    comment = 剧集评论数
    aliyun_video_id = 阿里云点播视频ID

  • 业务逻辑：获取video_episodes表中video_id=接收参数的数据，按照剧集数正序。

前端通过episodesId和剧集列表中的id匹配，获取对应播放的play_url和aliyun_video_id
