Django项目测试
为何要学习它

  • 代码可以运行起来 != 功能正常
  • 测试驱动开发 测试先行
  • 测试不仅能发现错误，还能预防错误
  • 自动化测试可提升工作效率

Django 使用的是内置于 Python 标准库中的 unittest 单元测试框架。 您也可以使用任何其他的 Python 测试 框架；
Django 提供了一个 API 和工具， 可以融合其它的单元测试框架。

Django 的单元测试使用 Python 标准库模块： unittest。 该模块定义使用基于类的方法测试。
在我们创建 Django 应用时， 默认已经帮我们生成了 tests.py 文件，

打开 .../system\tests.py 文件， 编写测试如下代码

accounts\tests.py
