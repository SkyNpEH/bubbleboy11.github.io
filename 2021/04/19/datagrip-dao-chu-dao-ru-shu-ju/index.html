DataGrip导出导入数据

发表于 2021-04-18 更新于 2021-06-07 分类于 数据库

DataGrip像navicat-工具-数据传输的功能
再win的
mysql,其他也一样
导出
1.右键导出的数据库，选择Export with mysqldump
Path to mysqldump: 选择C:/Program Files/MySQL/MySQL Server 8.0/bin/mysqldump.exe ，即 mysql bin 目录下的mysqldump文件

在自动生成的命令上加上-d参数,就可以仅仅导出表结构而不包含数据!如果想要导出的文件包含数据 , 加上-d参数可破…

3.out path 是你想要输出的文件目录，需要制定文件，例如 /Users/beliefx/数据/aa.sql

导入
1.右键导出的数据库，选择Retore with mysql
Path to mysql: 选择C:/Program Files/MySQL/MySQL Server 8.0/bin/mysql.exe ，即 mysql bin 目录下的mysql文件

  1. path to dump 是你想要导入的文件，例如 /Users/beliefx/数据/aa.sql

到Files栏目里面右键：Attach Directory选择sql所在目录
