141，检测链表是否有环：Linked List Cycle

链表不可能有多个环，因为一个节点不能有两个后继指针；
可行方案：
1、遍历是否能够出现 NIL 空指针；
2、使用 Set 存储已访问过的节点，如果当前待遍历的节点已存储在Set集合中，则证明有环；
3、快慢指针：慢指针一次走一步，快指针一次走两步，如果有环，两者一定会相遇；
def hasCycle(head):
slow = fast = head
while slow and fast and fast.next:
slow = slow.next
fast = fast.next.next
if slow == fast:
return True
return False
