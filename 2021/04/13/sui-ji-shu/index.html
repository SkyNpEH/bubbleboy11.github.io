python 随机数

发表于 2021-04-12 更新于 2021-04-14 分类于 python

python中生成随机整数、随机小数、0–1之间小数方法
随机整数：random.randint(a,b)，生成区间内的整数
随机小数：numpy.random.randn(5)生成5个随机小数
0-1随机小数：```random.random()``，括号中不传参
