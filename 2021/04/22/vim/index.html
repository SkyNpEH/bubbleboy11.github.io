VIM配置
a. 在文件打开时在末行模式下输入的配置（临时）
b.个人配置文件 ~/.vimrc
c. 全局配置文件 vim自带，/etc/vimrc
新建个人配置文件，i进入编辑
在配置文件中进行配置 显示行号: set nu

结论:如果针对同一个配置项,个人配置文件中存在,则以个人配置文件为准,如果个人配
置文件中不存在这一项,则以全局配置文件为准。

单词：以空格隔开的连续字符串
Vim编辑的文件中,每一行都有一个行号
行号从1开始,逐一递增
行号默认没有显示，可以按:进入命令模式,
输入 set nu显示行号,
输入 set nonu隐藏行号,

vim 三种模式 可以用ftp打开，

  • 交互命令模式: interactive Mode
    不能对文件直接编辑，可以输入快捷键删除行、复制行、上下左右移动光标、粘贴、跳转到指定行、撤销操作，这是运行Vim程序的打开文件的默认模式
    按字母键h,j,k,l或方向键↑↓←→光标在文本中上下左右移动一个字符
    数字健0或Home键 将光标定位到一行的开始位置
    美元键$或end键 将光标定位到一行的行尾
    ^ 光标移动到行首

gg 光标移动到第一行首行
G Shft+g光标移动到末行行首 go
7+ Shft+g 或 7gg 如果我们要跳转到第7行行首
行号+G或行号+gg跳转到指定行

ctrl+b 向上翻页 before
ctrl+f 向下翻页 after

w 一个单词一个单词地移动
u 撤销最近的修改，undo
4u 或者连按4次u 撤销最近4次修改
ctrl+r 取消撤销，重做之前的修改 redo
ctrl c cancel

光标定位的字符，按 x 键会删除这个字符，和插入模式下的Back键一样
按 x 之前输入数字 一次性删除多个字符
例如先按4再按x，光标所在字符为起点删除4个字符
d 剪切删除单词或者行，被删除的内容会被暂存在內存里，
连按两次d来删除光标所在的那一行
dd 也可以和数字配合 一次性删除多行
先输入2,再按下dd,就会删除从光标所在行开始向下的2行
dw 删除从光标当前字符开始到下一个空格的所有字符
3dw 或 d3w 删除3个单词

d0 删除从光标处到行首的所有字符
d$ 删除从光标处到行未的所有字符

yy 按两次y 复制光标所在行到内存中 yank
数字 yy ：以光标所在行为准（包含当前行），向下复制指定的行数

yw 复制一个单词
y$ 是复制从光标所在处到行末的所有字符
y0 是复制从光标所在处到行首的所有字符

p 内容会被粘贴到光标之后 paste
如果你用yy复制了一行,再用p来粘贴，那么这一行会被粘贴到光标所在行的下一行处

在p前面加上次数，将同样的內容粘贴多次
7p,表示粘贴7次

在交互模式下,将光标置于想要替换的字符上 按下r键 replace
接着输入你要替换的字符
rs表示替换当前字符为S
如果用大写的R,那就是切换到替换模式了，可一次性替换多个字符
左下角会显示– REPLACE –， Esc退出

—- 搜索及替换命令
/ 进入查找模式
/pattern：从光标开始处向文件尾搜索 光标就会转到文件中下一个查找到的匹配处
?pattern：从光标开始处向文件开头搜索pattern

:s/搜索的字符串/新字符串 替换光标所在行的第一个符合条件的字符串为新字符串
:s/搜索的字符串/新字符串/g 替换光标所在行的全部符合条件的字符串

:%s/搜索的字符串/新字符串 替换整个文档中每行的第一处符合条件的字符串
:%s/搜索的字符串/新字符串/g 替换整个文档所有匹配的字符串
%表示整个文件 g 全局global
vim打开的文件下把a替换成b （:%s /A/B/g）

:#,# s/旧字符串/新字符串/g替换文件中第#行到第#行的所有匹配的旧字符串为新字符串
:2,4 s/man/hero/g
1，$ s/p1/p2/g：将当前行中所有p1均用p2替代
n1,n2 s/p1/p2/g：将第n1至n2行中所有p1均用p2替代

n：在同一方向重复上一次搜索命令 查找下一个匹配项 next
N shift n：在反向上重复上一次搜索命令 在搜索结果中切换上一个匹配项

:r file_path 在光标处的下一行插人一个文件的内容

  • 编辑插入模式：对文件内容编辑insert
    按下面的键都可以进入
  • i 光标移动到当前光标所在字符前
  • I 光标移动到当前光标所在行行首
  • a 光标移动到当前光标所在字符后 after
  • s 删除光标所在的字符并进入
  • S 删除光标所在的行并进入
  • A 光标移动到当前光标所在行行尾
  • o 当前光标移动到当前光标所在行下面新增一行
  • O 当前光标移动到当前光标所在行上面新增一行
  • Esc 退出回到交互模式 escape

:w 保存，冒号进入命令模式 write
:w 路径 另存为

:wq 或 :x 没用修改退出文件/文件已修改，保存并退出
:q! 强制退出，刚才的修改操作不保存
:!command 外部命令 在vim调用外部命令
:!lsVim打开的文件的所在目录运行命令

:syntax on 着色显示 语法
:syntax off 着色关闭

:sp 默认横向分屏打开同一个文件 split
:sp file_path 默认横向分屏打开不同文件 split
:vsp 垂直分屏打开同一个文件 split
:sp file_path 默认横向分屏打开不同文件 split

分屏模式下的主要快捷键
Ctr+W然后再按Ctrl+w从一个 viewport移动光标到另一个 viewport

Ctr+W然后按h 移动光标到左边的 viewport
ctrl+W然后按j 移动光标到下方的 viewport
Ctr+W然后按k 移动光标到上边的 viewport
Ctr+W然后按l 移动光标到右边的 viewport
Ctr+W然后按+ 扩大当前 viewport
Ctr+W然后按- 缩小当前 viewport
Ctr+W然后按= 重新均匀分配各个 viewport的占比
Ctr+W然后按r 调换各个 viewport的位置
Ctr+W然后按R 反向调换各个 viewport的位置
Ctr+w然后按q或按C 或者 输入:quit或:close 关闭当前 viewport
Ctr+W然后按o， 或者 输入`:only 只保留当前所在 viewport,关闭其他 viewport
