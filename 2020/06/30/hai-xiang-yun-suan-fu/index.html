海象运算符

发表于 2020-06-29 更新于 2021-07-31 分类于 python

python3.8提供了一个新的运算符 - 海象运算符 Assignment Expresions ，
即 赋值表达式。

由一个 : 和 一个 = 组成，即：:= 。我们一般称作 walrus operator（海象运算符），因为它看起来就是一个海象旋转 90° 而成。

即一个变量名后跟一个表达式或者一个值，这个和赋值运算符 = 类似，可以看作是一种新的赋值运算符。
