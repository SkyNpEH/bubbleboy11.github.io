在django中使用redis

配置Redis缓存
之前配置的缓存都是使用Local Memory Cache即本地内存进行缓存的，Django重启之后就会消失，Redis作为backend进行了扩展和优化。

缓存应该考虑请求内容的格式是HTML还是json，请求是否包含参数等等问题，
这可以根据Redis的键值观察出来，
需要使用第三方库django-redis，
相同的请求多次访问，Redis也不会增加数据，只有进行了不同类型的请求才会增加数据，django-redis根据请求的不同类型生成key，以区分不同类型的请求。
