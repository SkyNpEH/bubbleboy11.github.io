加密方式

发表于 2020-08-20 更新于 2021-05-24 分类于 python

加解密算法，是对源数据 进行运算产生加密数据，以及反向过程，对加密数据反算出 源数据。

加解密算法 和 hash算法 不同点有：

  • 加解密算法 是可逆的，hash算法是不可逆的。
  • hash算法可以对很大的数据产生比较小的哈希值，而加密算法源数据很大，加密后的数据也会很大

加密方式

  • 对称加密：加密和解密是同一个密钥，速度快，一般对文件，AES, RC4, DES, 3DES, IDEA
  • 非对称加密： 加密和解密是不同的密钥，公钥加密——私钥解密，私钥签名——公钥验签，强度高速度慢，互联网 RSA (Rivest–Shamir–Adleman) DSA ECC
