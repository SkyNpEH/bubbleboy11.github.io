pycharm 快捷键 keymap

发表于 2020-08-20 更新于 2021-05-04

pycharm
alt + enter 导入引用包
比如 我们 引用 一个 python 模块的 random
显示 没有被引入
我们把鼠标 移到上面 然后 alt + enter 会提示我们 引用哪一个包

1.添加
多行全部选中
点击tab键
2. 删除
多行全部选中
shift +tab键

Ctrl + R 替换 若变量名同时也是某个字符串，则会把该字符串也给修改了
Ctrl + Shift + R 全局替换
Ctrl+Shitf+E 快速跳转至历史编辑过的代码位置(默认记录十条)

调出快速搜索框：两次Shift
全局查找快捷键默认：Ctrl+Shift+F （如果不可以就是与搜狗输入法的冲突了）
Getter和Setter封装：Alt+Insert
自动补全代码：Alt+/
撤销：Ctrl+Z
撤销返回：Ctrl+Shift+Z
复制本行代码到下一行：Ctrl+D
删除光标所在行的代码：Ctrl+Y
自动注释代码：Ctrl+Shift+/
测试接口的类：在要测试的上Ctrl + shift + t
快速展开/折叠全部方法：Ctrl + Shift + “ +/- “
向上插入一行空格：CTRL+Enter
向下插入一行空格：Shift+Enter
上端或下端插入一空行：shift+Alt+方向键
快速给代码加上组合语句try/catch等：CTRL+ALT+T
多行相同的代码一起修改或者删除：Ctrl+Shift+Alt 或者 按住鼠标的滚轮向下拉 也可！
运行编写的py文件：Ctrl+Shift+F10

eclispse快捷键
F5 进入函数逐行调试
F6 执行完一行代码
F8 执行到下一个断点处
ALT + 上/下 箭头 代码上下移动
ALT + 左/右 箭头 左–返回 - -右前进
ctrl + 左键 看源码 -
ctrl + / 注释代码
ctrl + home /end 一个单词一个单词的左右移动
shift + home/end 从头或者尾 选中一行代码
shift + tab 选中代码前调
ctrl + h 全局搜索
ctrl + o 列出当前文件有哪些类

追踪代码回跳到上次光标位置 ctrl+alt+左箭头
第三方库源
右上角有时候不是当前文件，右键一定是当前文件

断点所在的行一开始还没运行即将执行的语句
调试：慢动作播放
pycharm 右键debug
vscode 左边🐞按钮 或者菜单run-start debugging 在弹出的内容选择python file
step over 跳过子函数，单步，不关心函数内部，只看输出
step into 进入子函数
step into my code 直接跳到断点位置，不执行源码的子函数执行自己的
step out 跳出当前函数
run into 跳到下一段点
